FUNCIONAMIENTO DE LAS APLICACIONES WEB

Las aplicaciones web son programas que se ejecutan en un servidor
y se acceden a traves de un navegador en lugar de instalarse en un dispositivo local.

Funcionamiento

*Cliente y Servidor: La aplicacion web sigue un modelo cliente-servidor.
El cliente es el navegador que utiliza el usuario (como Chrome, Firefox, etc.),
y el servidor es donde reside la aplicacion.
*Solicitudes HTTP: Cuando abres una aplicacion web, tu navegador envia una solicitud HTTP al servidor.
*Procesamiento en el servidor: El servidor recibe la solicitud y procesa la informacion.
*Respuesta al cliente: Una vez que el servidor ha procesado la solicitud, envia una respuesta al cliente en forma de HTML, CSS y JavaScript.
Estos lenguajes son los que permiten que tu navegador muestre la aplicacion de manera interactiva.
*Interactividad:Gracias a JavaScript, las aplicaciones web pueden ser interactivas sin necesidad de recargar la pagina.
*Almacenamiento y estado: Muchas aplicaciones web utilizan almacenamiento local o cookies para mantener el estado del usuario entre sesiones,
lo que mejora la experiencia del usuario.
*Seguridad: Es importante que las aplicaciones web implementen medidas de seguridad como HTTPS
para proteger los datos que se intercambian entre el cliente y el servidor.

Cuales son las tecnologias para el desarrollo de las aplicaciones web

*HTML (HyperText Markup Language): Esencial para estructurar el contenido de las paginas web.
Define elementos como encabezados, parrafos, enlaces e imagenes.
*CSS (Cascading Style Sheets): Se utiliza para dar estilo a las paginas web, controlando colores, fuentes, margenes
y disposicion de los elementos.
*JavaScript: Lenguaje de programacion que permite anadir interactividad a las paginas web.
Facilita la manipulacion del DOM y la respuesta a eventos.
*Frameworks y Bibliotecas:
- *React: Biblioteca de JavaScript para construir interfaces de usuario.
- *Angular: Framework completo para desarrollar aplicaciones web dinamicas.
- *Vue.js: Framework progresivo que se puede integrar facilmente en proyectos existentes.
